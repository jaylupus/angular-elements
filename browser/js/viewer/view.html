Basic D3 Network Force Graph

This simple network diagram is based off Ray Villalobos's d3 tutorial. The key difference between this graph and the others in our directive store is that it takes one data file, instead of a separate file for nodes and edges. Here, the edges are represented by the "target" property of the node. 

{"nodes":[
{"name": "Grandpa Jones"},
{"name": "Papa Jim","target":[0]},
{"name": "Jane"},
{"name": "Jim Jr", "target":[1,2]},
{"name": "Big Steve", "target":[1,2]},
{"name": "Lil J", "target":[2,1]},
{"name": "That other kid"}
]}

Interactive Scatter Plot

Powered by the NVD3 library. The scatter plot is a good option for displaying data with multiple dimentions - in this case the petal width,petal height, and sepal length of the famous Iris data

By using the size and color of the points (sepal length and species type respectively) we are able to communicate a large amount of information in a small space. The interactive nature of this NVD3 directive lets users focus in on specific dimensions of the data.
