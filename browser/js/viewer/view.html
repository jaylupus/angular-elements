Data Visualization

Network Graphs

Basic D3 Network Force Graph

This simple network diagram is based off Ray Villalobos's d3 tutorial. The key difference between this graph and the others in our directive store is that it takes one data file, instead of a separate file for nodes and edges

D3 Force Network Graph - Bostock Example

This force network graph is based off this example from Mike Bostock. To supply your own data, provide a list of nodes and a list of edges. Note that the edges reference the nodes by their index position in the file, not their ID.

Network Force Graph with Node Images

This force network graph is based off this example from Mike Bostock. Here, the nodes are represented as images. In order to use this directive, add an image url on the "image" property of the node object.

The edge data in this data set also contains a value or weight to indicate the strength of the connection between the nodes.

Linear Charts & Plots

Discrete Bar Chart

Powered by the NVD3 library. The discrete bar charts is a good choice for comparing one dimensional data across several entities. In this case, we are comparing yearly profits (y axis) across the top earning companies in the US (spread across the x axis). You can toggle the data attribute informing the Y axis in the directive options.

Interactive Scatter Plot

Powered by the NVD3 library. The scatter plot is a good option for displaying data with multiple dimentions - in this case the petal width,petal height, and sepal length of the famous Iris data

By using the size and color of the points (sepal length and species type respectively) we are able to communicate a large amount of information in a small space. The interactive nature of this NVD3 directive lets users focus in on specific dimensions of the data.

Pie Chart with User Input

This simple chart is noteworthy in that it you can use it without providing a data set; simply enter your labels and values directly in the editor
